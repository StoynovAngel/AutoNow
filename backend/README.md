## Prerequisites

The following tools are required to build and run the application:

- Java Development Kit (JDK) version **21 or later**
- Docker Desktop
- pgAdmin 4 *(optional, used for database inspection and management)*

## Installation
Follow the steps defined in the repository to set up the development environment and run the application locally.
Define .env variables in a `.env` file based on the provided `.env.example` file.
